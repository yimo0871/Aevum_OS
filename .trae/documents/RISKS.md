## 风险详情

### R-002: LLM API 成本和延迟（高）

**描述**：Experience 生成和评估可能需要 LLM 调用，OpenAI API 的成本和延迟可能影响系统可用性。

**影响**：
- 每次经验生成可能需要 1-3 次 LLM 调用
- 10,000 条种子经验生成成本可能较高
- API 限流可能导致流水线阻塞

**缓解方案**：
1. Embedding 使用 `text-embedding-3-small`（低成本）
2. LLM 辅助评估设为可选（规则评估为默认）
3. 设计缓存层，相同上下文不重复调用
4. 种子数据生成使用模拟执行，不需要真实 LLM 调用

**触发条件**：单日 API 成本超过 $50，或 P95 延迟超过 5s。

**应急预案**：切换到本地模型（如 Ollama + llama-index）。
